Site and tools

All of the CSS and HTML — layout, fonts, colors (like 'em or not) — on this site are my own design. Simple, fast, and a careful selection of shades of blue. The only JavaScript is the pop-over side menu. No tracking. The site is built with Jekyll and hosted on Github Pages.
